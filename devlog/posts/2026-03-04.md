# hardware prices got me fucked up

After my sudden urge to ralph-loop everything, combined with the youtube algorithm deciding that my current interest is home-build servers, my current distraction over the past day has been speccing out a home server that I could also run open source LLMs on (and maybe play around with some reinforcement learning).

what the hell is going on

I'm not used to this shit - firstly, i've always been in the "just buy a laptop" camp so building a PC is new territory for me anyway, but even then i aint used to "hardware price go up". My previous hardware strategy was to just keep the same $2k budget and just get a way better laptop every 5 or so years (thanks Gordon).

I was on an auction site scrolling past $60 7th gen servers and then the next posting is 32Gb of ram for $800? wot? Then you're like "ok, so what kind of GPU would i need for local LLMs" (turns out its all about VRAM, you learn something new every day) - nek minute a 5090 costs more than a second hand car.

I don't have anything hugely interesting to say about this. But it kinda freaks me out. Its unfamiliar territory. To try and capture the feeling - never in my life of playing with technology, has the core components of a new system felt so out of reach. Its a new feeling to me to say "I want to experiment with this" and its prohibitively expensive. Now ok sure - i could technically afford to build a rig, but a) im not sure it would be a "meaningful" capability, compared to API frontier models - like idk, but im probably not pulling off ralph loops with Quen or something and b) its not a "discretionary" purchase.

I should also point out there are also other solutions to this - it is possible to rent GPU / server access in the short term. But I like owning things. Its a hurdle I had to jump in the beginning of all this AI shit, when I was like "what, I have to pay for API credits?" - "can't I just run this on my own hardware". Its so against everything I've learned growing up playing with open source software that run on normal cpus (or micros I guess).

It also highlights to me how spicy it is to be relying on these APIs. I've already run into a small example of this. Currently we pay for Claude Max, and use claude code. However, I've been hearing good things about opencode, and thought I'd try it. You might have heard the drama recently about this - up until a week or so ago (idk, im not a journalist) it was possible to use your claude subscription with opencode (great - i already pay for that, so we can just try it out and see). But now Dario says no - only claude code can use the claude subscription. So anyway, now I have a codex subscription and claude max.

It feels yucky to be so blatantly stuck inside a battle for market share. At least google had the decency to sell our private info instead of charging us money, so it felt like a good deal if you don't think about it too hard.

so yea - thats my existential crisis for the day

see you tomorrow.
